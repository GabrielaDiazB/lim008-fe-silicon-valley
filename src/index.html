Mis Proyectos

Cada proyecto realizado requirió el aprendizaje de herramientas nuevas con las que no había trabajado antes. Fue así que en estos tres proyectos utilicé, además de lenguaje javascript, html y css, manipulación de datos del tipo JSON, Firebase, base de datos, autenticación de data, callbacks, NodeJs, promosas, entre otros.

Manipulación de data tipo JSON

En el primer proyecto, llamado "Data Lovers", se realizó una página web donde se pudiese mostrar información y datos estadísticos de los campeones/personajes del juego de tipo MOBA "League of Legends". La información con la que se decidió trabajar cumple con la solicitud de una serie de usuarios entrevistados para poder visualizar de una forma más legible, simple y amigable los datos de los personajes con los que desean jugar.

Red Social

En el segundo proyecto, llamado "Red Social: Agogo", se creó una red social que cumpla con la condición de poder crearse una cuenta o iniciar sesión con otras cuentas existentes como Facebook, Twitter o Gmail. Además de ello, la aplicación web te permite realizar posts de forma pública o privada así como editarlos o poder eliminarlos. El tema de la red social fue obtenido tras un breve estudio con usuarios para saber qué tipo de red social era la que "les hacía falta".

Creación de una Librería

Por último, el tercer proyecto fue llamado "Markdown Links". Consistió en la creación de una libreria instalable que permitiera extraer, analizar y valide los links de todos los archivos del tipo Markdown. Basta con instalarlo e ingresar una ruta para que la librería lo recorra y muestre cuantos de estos links son válidos, están rotos, son repetidos, etc.
